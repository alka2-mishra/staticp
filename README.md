# Simple Static Website

A basic responsive static website built using **HTML** and **CSS**.  
This project contains multiple pages including a Home page, About page, and Contact page.

---

## Features

- Responsive Navigation Bar
- Home, About, and Contact Pages
- Hero Section with Image
- Service Cards Section
- Contact Form
- Footer Section
- Responsive Design using CSS Media Queries

---

## Technologies Used

- HTML5
- CSS3
